Equal Height

Can be used with elements other than cards.

  1. Assign .js-bins to grid container.
  2. Set data-bin-viewport on container to a value from _mq-check.scss. Height calculations will be removed for that breakpoint and smaller.
  3. Add class of .bin on elements where equal height should be set.
  4. Add data-bin-shivto element within .bin. This appends element where height is applied.
